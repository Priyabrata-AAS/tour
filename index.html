üìç 6. LIVE LOCATION SHARING (REALITY CHECK)

What CAN Be Done:

  • Driver manually shares live location via WhatsApp
  • System sends WhatsApp reminders before location expiry
  • Customer tracks in real-time through WhatsApp

WhatsApp Limitations (IMPORTANT):

  • Live location valid for maximum 8 hours
  • Must be manually re-shared for longer trips
  • No API to auto-extend location sharing
  • WhatsApp privacy policy prevents automation

Recommended Approach:

Trip Duration System Action
‚â§ 8 hours One location share sufficient
> 8 hours Reminder sent 2 hours before expiry:
"Your live location will expire in 2 hours. Please re-share to keep customer updated."
